# del_mlxenv
Frees the MiniLibX environment object.

#### Parameters
- `t_mlxenv *m_env`: Pointer to the MiniLibX environment object to be freed.

#### Returns
- None

#### Example Usage
```c
del_mlxenv(env);
env = NULL;
```
